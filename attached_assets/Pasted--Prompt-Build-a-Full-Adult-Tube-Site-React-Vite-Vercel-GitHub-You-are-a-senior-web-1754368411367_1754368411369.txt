### 🎯 Prompt: Build a Full Adult Tube Site (React + Vite + Vercel + GitHub)

You are a senior web engineer. Build a **high-performance adult tube-style website** using **React + Vite**, hosted on **Vercel** with content versioned in **GitHub**. The site should feel like Pornhub or XVideos: infinite scrolling, autoplay previews, embedded video playback, smart categorization, and ad injection zones.

---

### 🧱 Tech Stack

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS + shadcn/ui + Framer Motion
- **Markdown/Content**: `index.md` files per video
- **Routing**: React Router or File-based routing
- **Hosting**: GitHub + Vercel
- **Playback**: Embedded iframes (Xerotica, RedGIFs, etc.) or native `<video>` tag

---

### 📁 Folder + Content Structure

```
data/
  mila/
    wild-flower/
      index.md
```

Each video lives in `data/<model>/<slug>/index.md` and includes:

- Metadata: title, description, tags, duration, source
- Cover thumbnail image (referenced inline)
- Video iframe or source URL (Xerotica, RedGIF, custom)
- Optional preview GIF or autoplay MP4 (for hover previews)

---

### 📽️ Tube-Site Features

#### 🔥 Home Page

- Video Grid with title, tags, duration, view count, model
- Autoplay preview on hover (MP4 or GIF)
- Filter by Category, Tag, Model, Duration
- “Trending”, “Newest”, “Top Rated” sections

#### 🎞️ Video Page

- Embedded player with title + description
- Tags + model profile
- View count, likes, related videos
- Ad slots (Top, Mid, Bottom)
- Social share buttons (Telegram, WhatsApp, etc.)

---

### 📢 Ad Integration

Inject ads in 3 positions per video page:

1. ✅ **Top AdSlot** – below player or above title  
2. ✅ **Mid AdSlot** – between description and tags  
3. ✅ **Bottom AdSlot** – before related videos  

Supports:

- Banner ads (image/HTML)
- Cam site embeds
- Smartlinks / Native ads
- React components like:

```tsx
<AdSlot position="top" />
<AdSlot position="middle" />
<AdSlot position="bottom" />
```
